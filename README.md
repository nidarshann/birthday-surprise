birthday-26
==========
Minimal Next.js project (5 pages) that displays "26 years ago" across every page.

How to use
----------
1. Install dependencies: npm install
2. Run dev: npm run dev (site available at http://localhost:3000)
3. Build & start: npm run build && npm start

Deploy to Vercel
----------------
- Create a new GitHub repo, push this project, then import into Vercel.
- Alternatively, use Vercel CLI: vercel login; vercel --prod
